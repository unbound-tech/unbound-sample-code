    - Ethereum
1. Certificates
    - C
    - Java
1. Code Signing
    - Windows Office
1. Derivation
    - C
1. Encryption
    - C
    - Java
1. Key Management
    - C
    - Java - PKCS#11
1. Passwords
    - C
1. Signing
    - C
    - Java
1. Tokenization
    - Java - Using Tokenization with FPE 
1. Wrapping
    - C
    - Java - PKCS#11
